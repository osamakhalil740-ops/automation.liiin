   # Edit .env with your JWT_SECRET and GEMINI_API_KEY

   npx playwright install chromium
   npx prisma generate
   npx prisma db push

   pm2 start ecosystem.config.js

   http://your-server-ip:3000

   - Create keywords
   - Add comment templates
   - Set engagement filters
   - Enable system

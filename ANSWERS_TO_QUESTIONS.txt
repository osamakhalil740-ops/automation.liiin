### OPTION 1: PM2 (Recommended for Production)

**One-Time Setup:**
\\\powershell
# Install PM2 globally
npm install -g pm2

# Start both web server AND worker automatically
cd C:\Users\lenovo\Downloads\clonelink
pm2 start ecosystem.config.js

# Save configuration
pm2 save

# Setup auto-start on server reboot
pm2 startup
\\\

**What This Does:**
✅ Starts web server on port 3000
✅ Starts worker automatically
✅ Both auto-restart if they crash
✅ Both auto-start when server reboots
✅ ZERO manual intervention needed

**Managing:**
\\\powershell
pm2 status          # Check status
pm2 logs            # View logs
pm2 restart all     # Restart everything
pm2 stop all        # Stop everything
pm2 monit           # Live monitoring
\\\
